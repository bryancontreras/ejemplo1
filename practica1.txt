******Trabajar en remoto usando HTTPS****

Comandos:

$git remote add origin URL HTTPS

$git push origin master/main       ***GIT PUSH envia el guardado en el repositorio local, los envía al repositorio remoto***

Para trabajar en RAMAS

$git branch RAMA                   ***GIT BRANCH nos crea una rama. 

$git ckeckout RAMA	           ***GIT CHECKOUT nos envía a la rama seleccionada

$git status
$git add archivo.txt
$git commint -m "texto"
git log

GitHub, subir la rama

$git push RAMA

Para copiar un repositorio:
$git clone URL

/////GIT FETCH  actualizaciones del repositorio remoto al repositorio local
////GIT PULL  actualza los cambios hechos en el repositorio remoto y los guarda en el repositorio local y en nuestro directorio de trabajo
